## 1. 개요

본 문서는 FPGA 기반 FM/DMB 디지털 리피터 프로토타입 1set 제작에 필요한 **전체 개발 비용을 Phase별로 상세 견적**한 자료이다.

### 개발 범위

| 항목 | 내용 |
|------|------|
| 대상 주파수 | FM 88~108 MHz, DMB 174~216 MHz (VHF Band III) |
| 변조 방식 | FM: WBFM (Analog), DMB: OFDM (DAB/T-DMB) |
| 출력 | FM: 10mW 이하 (특정소출력), DMB: 10mW 이하 |
| 처리 방식 | FPGA 기반 디지털 CIC + FIR + ISOP 필터 + AGC |
| FPGA 칩 | Xilinx Kintex-7 XC7K325T-2FFG900I |
| PCB | 8층, FR-4/Hi-Tg, 임피던스 제어 |
| 프로토타입 수량 | 3set (1set: 엔지니어링 검증, 1set: 튜닝, 1set: 인증/전시) |
