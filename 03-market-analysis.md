# 국내 FM/DMB 리피터 시장 분석 보고서

> 2025년 기준 한국 재난방송 중계기 시장 현황, 경쟁 구도, 진입 전략

---

## 1. 시장 개요

### 1.1 시장 정의

**DMB·FM 리피터(중계기)** 는 방송공동수신설비(헤드엔드)에서 전송된 FM 라디오(88~108MHz) 및 지상파 DMB(174~216MHz) 신호를 수신하여 지하주차장, 지하상가, 터널 등 **전파 음영 지역**에 무선 재전송하는 **특정소출력 무선기기**다.

### 1.2 시장 규모 추정

| 구분 | 내용 |
|---|---|
| 국내 터널 수 | **약 5,380개** (도로·철도·도시철도) |
| FM 수신불량 터널 | 약 3,380개 (전체 68.2%) |
| DMB 수신불량 터널 | 약 3,400개 (전체 68.4%) |
| 연간 리피터 수요 추정 | **약 3만~5만대** |
| 리피터 단가 (주력) | 33~55만원 |
| FM/DMB 프로세서 단가 | 12~120만원 |
| **전체 연관 시장 규모** | **약 200~400억 원** (리피터 + 프로세서 + 전원공급기 + 케이블 + 안테나 + 시공) |

### 1.3 핵심 성장 동인

| 동인 | 내용 |
|---|---|
| **법적 의무화** | 방송통신발전기본법 제40조의3 — 터널·지하공간 중계설비 설치 의무화 |
| **재난방송 채널 확대** | 2015년 8월 T-DMB 재난방송 추가 지정 (기존 FM만 → FM + DMB) |
| **정부 지원 확대** | 2024년 20대 → 2025년 60대로 지원 규모 확대 |
| **터널 200m 기준 법제화 (2025)** | 200m 이상 터널 전체(약 80%) 재난방송 설비 설치 우선 지원 |
| **신규 아파트 공급** | 공동주택 지하주차장 재난방송 중계설비 의무화 |

## 2. 제품군 분류

### 2.1 시스템 계층도

```
안테나 → [수신공용기] → [프로세서] → [리피터] → [안테나/방사]
                              ↑
                        전원공급기/전원삽입기
```

### 2.2 제품군별 가격대

| 제품군 | 역할 | 가격대 | 주요 모델 |
|---|---|---|---|
| **리피터 (Repeater)** | 신호 수신 → 증폭 → 무선 재방사 | **33~110만원** | 전체 제품 리스트 참조 |
| **프로세서 (Processor)** | FM/DMB 채널별 신호 레벨 조정, AGC | **12~120만원** | SKC-DFP2000, WJFM-100EQ |
| **전원공급기/삽입기** | 동축케이블로 RF+전원 중첩 전송 | **4~20만원** | DPI-40, KIPS-550D |
| **확장 안테나** | 리피터 방사 범위 확장 | **5만원** | DTX-A |
| **대용량/터널형 시스템** | 광링크 기반 터널 전체 커버 | **1,000만원↑** | FRTek 터널형 시스템 |

## 8. 시장 트렌드 및 전망

### 8.1 단기 트렌드 (2025-2026)

| 트렌드 | 영향 |
|---|---|
| 터널 재난방송 의무화 본격 시행 | 대형 시스템 수요 급증, 광링크 방식 증가 |
| 정부 예산 지원 확대 | 공공 발주 증가 |
| 아파트 분량 물량 회복 | 신규 지하주차장 리피터 수요 |
| 가격 경쟁 심화 | 33만원대 제품 등장, 추가 하락 압력 |

### 8.2 중장기 트렌드 (2027-2030)

- **비상방송 기능 통합**: 단순 중계기 → 재난경보 게이트웨이로 진화
- **IoT/NMS 기능 탑재**: 원격 모니터링, SNMP, 자가 진단
- **저전력화**: 태양광 + 배터리 구동 옵션
- **디지털 전환**: 아날로그 증폭 방식 → FPGA/SDR 기반 디지털 신호 처리

### 8.3 예상 시장 규모 추이

```
2024년: 약 200~300억 원
2025년: 약 250~350억 원 (터널 법제화 효과)

2027년~: 연 5~10% 성장 → 시장 성숙기 진입
```

### 9.1 현 시장의 문제점 (갭 분석)

| 문제점 | 설명 |
|---|---|
| **제품 동질화** | 모든 제품이 동일 스펙, 동일 가격 — 차별화 포인트 없음 |
| **기술 혁신 부재** | FPGA 디지털 중계 기술(40ch 동시 처리 등)이 전혀 반영되지 않음 |
| **저가 경쟁 심화** | 33만원까지 하락, 마진율 10~15% 추정 |
| **유통 종속** | 소수 채널(MATV, 태화텔레콤)에 집중 — 제조사 브랜드 인지도 낮음 |
| **A/S 표준화 부재** | 제조사별 A/S 정책 불투명 |

### 9.2 권장 진입 포지셔닝: 디지털 프리미엄

```
현행 시장 제품: [아날로그 증폭기 수준]
    - 단순 BPF + Amp + 안테나
    - FM/DMB 단순 증폭
    - 가격 33~40만원

제안 진입 포지셔닝: [디지털 DSP 기반 스마트 리피터]
    - FPGA 기반 채널별 디지털 필터링 + AGC
    - FM 40ch + DMB 6ch 동시 처리
    - 비상방송 연동, NMS 원격 모니터링
    - 가격 60~90만원 (프리미엄)
```

### 9.3 경쟁 우위 확보 방안

| 전략 | 내용 |
|---|---|
| **디지털 기술 차별화** | FPGA 기반 정밀 채널 필터링 → 기존 아날로그 대비 음질·화질 우위 |
| **NMS/원격 관리** | SNMP 기반 원격 모니터링 — 대형 현장에서 강력한 메리트 |
| **인증 조기 확보** | KC 인증, RAPA 성능평가시험 선제 통과 |
| **B2G 공공 조달** | 터널형 시스템(광링크+비상방송)으로 공공 시장 공략 |
| **온라인 + 오프라인 병행** | 초기 온라인 인지도 확보 → 오프라인 대리점 확장 |
