# 제조사 제품 분해 분석 — 역설계 BOM 비교표

> 시중 판매 중인 FM/DMB 리피터의 추정 내부 구조 및 부품 BOM 역추적 분석

---

## 1. 분석 개요

본 문서는 시중에 판매 중인 국산 FM/DMB 리피터 제품들의 내부 구조를 **역설계(reverse engineering)** 방식으로 추정 분석한다. 실제 제품을 분해하지 않은 상태에서 공개된 제품 사양, 유사 RF 제품의 분해 사례, 일반적인 RF 증폭기 회로 구조, 시판 RF IC 데이터시트를 기반으로 추론한 결과다.

> **⚠️ 한계**: 실제 제품 분해 없이 추정한 데이터이므로 실제 BOM과 차이가 있을 수 있음. 주요 IC 선정은 시장에 유통 중인 범용 RF 부품과 유사 회로의 분해 사례를 기반으로 추론.

---

## 2. 제품군별 내부 구조 추정

### 2.1 보급형 (33~45만원대) — 아날로그 BPF + MMIC 증폭 구조

#### 추정 블록도

```
[F-Type Input] → [BPF (FM)] ←┐
                               ├→ [Combiner] → [MMIC Amp] → [ALC] → [Helical ANT]
[F-Type Input] → [BPF (DMB)] ←┘
                               ↓
                          [Power Supply (DC-DC)]
                               ↓
                          [DC Input (5~24V)]
```

## 5. 아날로그 방식 vs FPGA 디지털 방식 BOM 비교

| 항목 | 현행 아날로그 (보급형) | 현행 아날로그 (중급형) | FPGA 디지털 방식 (제안) |
|---|---|---|---|
| **주요 IC** | MMIC×3, Op-Amp×1 | MMIC×3, Digi-Atten×1 | FPGA×1, ADC×1, DAC×1, LNA×2 |
| **필터** | SAW BPF×2 (고정) | SAW BPF×2 (고정) | 디지털 필터 (프로그래머블) |
| **채널 수** | FM+DMB 단순 증폭 | FM+DMB 단순 증폭 | FM 40ch + DMB 6ch |
| **BOM 부품비** | **$5~12** | **$12~20** | **$90~140** |
| **PCB 비용** | 2층 FR-4: $2~5 | 2층 FR-4: $3~5 | 6층 FR-4 + 임피던스: $50~100 |
| **조립 난이도** | 낮음 (수삽+SMT) | 낮음 | 높음 (BGA, 0.5mm pitch) |
| **개발 비용** | $5K~20K | $10K~30K | **$50K~150K** (FPGA 펌웨어 포함) |
| **제조 원가** | **$16~23** | **$27~35** | **$250~400** |
| **소비자가 (목표)** | 33~45만원 | 50~60만원 | **60~90만원** |
| **BOM/소비자가 비율** | 3~4% | 4~5% | **20~25%** (정상적인 전자기기 비율) |

### 5.1 시사점

| 항목 | 내용 |
|---|---|
| **현행 제품은 BOM 비중이 극히 낮음** | 소비자가의 3~5%만 부품비 — 나머지는 유통/마케팅/시공 마진 |
| **FPGA 방식은 BOM 비중이 정상적** | 20~25% — 일반 전자기기와 유사한 구조 |
| **FPGA 방식의 진정한 가치는 소프트웨어** | 펌웨어 업데이트로 기능 추가 가능 → 지속적 수익 모델 |
| **기존 제품의 기술적 취약점** | 온도 드리프트, 부품 편차, 채널 간 간섭 — 디지털 방식이 모두 해결 가능 |

## 6. 주요 결론

1. **현행 제품의 BOM 부품비는 1~2만원 수준**으로 극히 낮음. 제품 간 기술적 격차가 거의 없음.

2. **유통 마진이 가격의 90% 이상**을 차지. 제조사 마진은 생각보다 높지 않을 수 있음 (납품가 15~25만원 추정).

3. **FPGA 디지털 방식은 BOM에서 10~15배 비싸지만**, 제공하는 기능적 우위 (다채널 처리, 정밀 AGC, 원격 모니터링, 소프트웨어 업데이트)가 차별화를 만들어낼 수 있음.

4. **핵심 경쟁력은 하드웨어가 아니라 소프트웨어/펌웨어**. FPGA IP 코어 (CIC, FIR, AGC)의 설계 품질이 제품 성능을 결정.
