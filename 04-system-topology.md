# FM/DMB 재난방송 중계 시스템 — 최상위 시스템 구성도

> 리피터(중계기)를 포함한 전체 방송공동수신설비의 계층적 시스템 구조

---

## 1. 시스템 개요

FM/DMB 재난방송 중계 시스템은 **지상파 방송사**가 송신소를 통해 송출하는 전파를 **건물 옥상 안테나**로 수신하여 **지하주차장, 터널, 지하상가** 등의 전파 음영 지역에 재전송하는 전체 설비를 의미한다.

## 2. 주요 장비별 상세

### 2.1 Layer 1: 수신 안테나부

| 장비 | 주파수 | 임피던스 | 설치 위치 | 비고 |
|---|---|---|---|---|
| **FM 안테나** (GP type) | 88~108 MHz | 75Ω | 옥상 / 옥외 | Yagi, Dipole, GP형 |
| **DMB 안테나** (GP type) | 174~216 MHz (Band-III) | 75Ω | 옥상 / 옥외 | VHF High 대역 |
| **UHF TV 안테나** | 470~770 MHz | 75Ω | 옥상 | 지상파 DTV 수신 |
| **위성 안테나** | 950~2150 MHz | 75Ω | 옥상 | 위성방송 수신 |
| **안테나 지지대 / 마스트** | — | — | 옥상 | 접지 포함 |

### 2.2 Layer 2: 헤드엔드부 (수신공용장치부)

| 장비 | 기능 | 주요 기술사양 |
|---|---|---|
| **FM 수신증폭주장치** | FM 대역 저잡음 증폭 | 88~108MHz, NF<3dB, Gain 20~40dB |
| **DMB 수신증폭주장치** | DMB 대역 저잡음 증폭 | 174~216MHz, NF<3dB, Gain 20~40dB |
| **FM 프로세서 (Signal Processor)** | FM 채널별 레벨 조정, AGC | 12채널 / 40채널, 채널별 독립 AGC, 출력레벨 조정 |
| **DMB 프로세서** | DMB 채널 레벨 조정, AGC | 대역 통합 또는 채널별 조정 |
| **FM/DMB 통합 프로세서** | FM + DMB 동시 처리 | NDFP-2000 등, Dual Band 설계 |
| **채널 혼합기 (Combiner)** | TV + FM + DMB 신호 합성 | 삽입손실 최소화, 분리도 > 30dB |
| **디지털수신공용기** | AM/FM/T-DMB 통합 + 광출력 | 광 2포트, NMS 모니터링 |

### 2.3 Layer 3: 전송선로부

#### 3.1 케이블 방식

| 방식 | 케이블 종류 | 임피던스 | 전송 거리 | 특징 |
|---|---|---|---|---|
| **동축케이블** | 5C-2V / 7C-2V / 10C-2V | 75Ω | ~300m | 저렴, 설계 간단 |
| **광케이블** | SM (단모드) 1Core | — | ~20km | 장거리, 무손실, 대용량 |
| **원방급전 (Power over Coax)** | 동축케이블 | 75Ω | RF+DC 중첩 | 별도 전원공사 불필요 |

#### 3.2 선로 장비

| 장비 | 기능 | 비고 |
|---|---|---|
| **간선분배기 (Splitter)** | 신호 분배 (2/4/6/8 way) | 75Ω, 분배손실 3~12dB |
| **간선증폭기 (Line Amp)** | 신호 레벨 보상 증폭 | DIA-40 등, 필터방식 |
| **전원삽입기 (Power Inserter)** | 동축에 DC 전원 중첩 | DPI-40, CHT-PI-12, SKC-LPS2000 |
| **DC 전원공급기 (Power Supply)** | AC→DC 변환 | DC 5V/7V/12V/24V/48V |
| **신호분배/전원공급 복합기** | 전원삽입 + 분배 일체 | RFPID-40PD, DPS-40 |

#### 3.3 광전송 장비 (터널/대규모 시스템)

| 장비 | 기능 |
|---|---|
| **광송신기 (Optical Transmitter)** | RF 신호 → 광신호 변환 (1550nm) |
| **광수신기 (Optical Receiver)** | 광신호 → RF 신호 복원 |
| **광증폭기 (Optical Amplifier)** | 광신호 증폭 (EDFA) |
| **광분배함 (FDF)** | 광케이블 접속 및 분배 |
| **광스위치** | 광경로 절체 (이중화) |

### 2.4 Layer 4: 중계기부 (Repeater Section)

> **※ 이 Layer가 본 프로젝트의 핵심 대상 장비**

#### 4.1 중계기 유형별 분류

| 유형 | 적용 장소 | 특징 |
|---|---|---|
| **노출형 리피터** | 지하주차장 천장, 벽면 | 가장 일반적, 안테나 일체형 |
| **매입형 리피터** | 세대단자함 내부 | WHR-IDF05 등, 소형 |
| **대용량 리피터** | 넓은 지하공간 | DTX-40 (7대 직렬연결) |
| **원방급전형 리피터** | 전원 인입 어려운 곳 | 동축으로 RF+전원 수신 |
| **광중계기 (터널형)** | 터널, 지하철 | 광링크 + Remote Unit |
| **복스형 (방수형)** | 옥외, 반지하 | KI-DMBR(B), 방수 구조 |

#### 4.2 중계기 표준 기술 규격 (공통)

| 항목 | FM | T-DMB |
|---|---|---|
| 주파수 대역 | 88 ~ 108 MHz | 174 ~ 216 MHz |
| 입력 임피던스 | 75Ω (F-Type) | 75Ω |
| 방사 안테나 | 무지향성 (Helical, λ/4) | Helical, λ/4 |
| RF 출력 | 100~105 dBμV | 100~105 dBμV |
| 사용 전원 | DC 5~24V (모델별 상이) | |
| 소비 전류 | 단일 70mA / 겸용 130~270mA | |
| 중계 반경 | 25~70m (모델별 상이) | |
| AGC/ALC | 자동 이득 제어 기능 내장 | |

#### 4.3 중계기 주변 기기

| 장비 | 역할 |
|---|---|
| **확장 안테나 (DTX-A 등)** | 중계기 방사 범위 확장 |
| **선로형 증폭기 (HSDM-101 등)** | 리피터 간 신호 보상 (외부전원 불필요) |
| **리피터용 분배기 (DY-200TDPD)** | 다중 리피터 연결용 6분배기 |
| **멀티밴드 안테나** | FM + DMB 통합 안테나 |

### 2.5 Layer 5: 최종 수신 단말

| 단말 | 수신 가능 서비스 |
|---|---|
| **차량용 FM 라디오** | FM 재난방송 청취 |
| **DMB 휴대폰** | T-DMB 방송 시청 (지상파 3사) |
| **차량용 DMB** | T-DMB 방송 시청 |
| **휴대용 라디오** | FM 재난방송 청취 |
| **내비게이션 (DMB 내장)** | T-DMB 방송 시청 |
| **비상방송 스피커** | FM 멘트방송 수신 |

## 3. 적용 장소별 시스템 구성

### 3.1 아파트 지하주차장 (일반형) — RF 분산 방식

```
                    지상 (옥상)
             ┌───────────────┐
             │  FM ANT +     │
             │  DMB ANT      │
             └───────┬───────┘
                     │ 동축케이블 (75Ω)
                     ▼
             ┌───────────────┐   헤드엔드
             │  FM Processor │   (장치함 내)
             │  DMB Processor│
             │  + Combiner   │
             └───────┬───────┘
                     │ TV+FM+DMB 신호 합성
                     ▼
             ┌───────────────┐   지하 주장치함
             │  전원삽입기   │
             │  (Power       │
             │   Inserter)   │
             └───────┬───────┘
                     │ 동축케이블 (RF+DC 전원 중첩)
        ┌────────────┼────────────┐
        ▼            ▼             ▼
 ┌──────────┐ ┌──────────┐ ┌──────────┐
 │ FM/DMB   │ │ FM/DMB   │ │ FM/DMB   │  ← 최대 7대 직렬 연결
 │ 리피터   │←│ 리피터   │←│ 리피터   │
 │  (안테나 │ │  (안테나 │ │  (안테나 │
 │   일체형)│ │   일체형)│ │   일체형)│
 └────┬─────┘ └────┬─────┘ └────┬─────┘
      │            │            │
      ▼            ▼            ▼
   FM/DMB 전파 방사 (25~35m 반경)
```

**적용 대상**: 공동주택 / 아파트 지하주차장

**주요 자재**: FM/DMB 리피터 + 전원공급기 + 전원삽입기 + 동축케이블 + 분배기 + 무지향성 안테나

### 3.2 터널/지하철 — 광전송 방식 (Master-Slave)

```
                    지상
             ┌───────────────┐
             │  FM ANT +     │
             │  DMB ANT      │
             └───────┬───────┘
                     ▼
             ┌───────────────────┐   관리동
             │  디지털수신공용기  │   주장치(MU)
             │  (AM/FM/T-DMB)    │
             │  + 비상방송장치   │
             │  + 광송신기       │
             │  + NMS 모니터     │
             └───────┬───────┘
                     │ 광케이블 (SM 1Core)
        ┌────────────┼────────────┐
        ▼            ▼            ▼
 ┌──────────┐ ┌──────────┐ ┌──────────┐
 │ 광보조   │ │ 광보조   │ │ 광보조   │  자장치(RU)
 │ 중계기   │ │ 중계기   │ │ 중계기   │  Remote Unit
 └────┬─────┘ └────┬─────┘ └────┬─────┘
      │            │            │
      ▼            ▼            ▼
 ┌──────────┐ ┌──────────┐ ┌──────────┐
 │ 누설동축 │ │ 누설동축 │ │ 누설동축 │  LCX
 │ 케이블   │ │ 케이블   │ │ 케이블   │  Leaky Coax
 └──────────┘ └──────────┘ └──────────┘
      │            │            │
      ▼            ▼            ▼
   터널 내 FM/DMB 전파 방사 (전 구간)
```

**적용 대상**: 도로 터널, 철도 터널, 지하철 역사

**특징**:
- 하나의 **수신 안테나**로 다수 터널 서비스 가능
- 광케이블로 장거리 무손실 전송 (최대 20km)
- 광 **Bypass 기능**으로 중간 장비 고장 시에도 나머지 정상 동작
- **비상방송** 연동: 직접 멘트방송 / USB 녹음파일 / 내장 메모리 호출
- 원격제어감시 (NMS) 기본 탑재
- 배터리 내장: 정전 시에도 동작
- FM 40CH + DMB 2CH 지원

**주요 장비**: 주장치(MU) + 자장치(RU) + 광송수신기 + 누설동축케이블(LCX) + 선로증폭기 + 비상방송장치 + NMS 서버

   ┌──────────┐  ┌──────────┐  ┌──────────┐

> **⚠️ 논란 사항** (2025년 기준):
> - **겸용 찬성**: NFPC-505 5조, 공사비 절감, DAS 활용
> - **분리 주장**: 임피던스 차이(50Ω vs 75Ω), 단일 장애점(SPOF), 소방 본연 기능 저해
> - 실무적으로는 **동축케이블만 공유**하거나 완전 **분리 구축**하는 경우로 양분됨

## 9. 장비 분류 요약표 (전체 시스템)

| 계층 | 장비군 | 핵심 장비 | 비고 |
|---|---|---|---|
| **L1** | 안테나 | FM / DMB / TV / 위성 안테나 | 수신 신호 품질이 전체 성능 결정 |
| **L2** | 헤드엔드 | 수신증폭기, Processor, 혼합기 | 신호 정제 및 레벨 조정 |
| **L3** | 전송선로 | 동축·광케이블, 분배기, 증폭기, 전원삽입기 | 신호 전달 매체 |
| **L4** | 중계기 | **FM/DMB 리피터** | **본 프로젝트의 핵심 장비** |
| **L4a** | 주변기기 | 확장안테나, 선로증폭기, 분배기 | 리피터 보조 |
| **L3b** | 광전송 (터널) | 광송수신기, MU, RU, LCX | 터널 전용 |
| **—** | 관리 | NMS 서버, 비상방송장치 | 통합 관제 |

> **※ Layer 4 (중계기/리피터)** 가 본 프로젝트의 핵심 개발 대상이며, 다른 모든 장비는 리피터가 정상 동작하기 위한 **인프라 시스템**으로 이해해야 함.
